# Seed Database с фейковыми пользователями

## Описание

Seed скрипт создает 10 тестовых пользователей (5 женщин и 5 мужчин) для быстрого тестирования приложения. Каждый пользователь имеет:

- Профиль с отображаемым именем, биографией, полом и городом
- 2-3 фотографии из Unsplash (CC0 лицензия)
- Дату рождения для расчета возраста

## Использование

### 1. Убедитесь, что БД запущена

```bash
# Если используете Docker Compose
docker-compose up -d
```

### 2. Запустите seed скрипт

```bash
npm run seed
```

Скрипт автоматически:
- Подключится к БД по переменной окружения `DATABASE_URL`
- Очистит таблицы `profile_photos`, `profiles`, `users`
- Создаст 10 новых пользователей с профилями и фотографиями

## Фотографии

Все фотографии загружаются напрямую с Unsplash API:
- Публичные изображения с лицензией CC0
- Разные изображения каждый раз благодаря параметру `seed`
- Размер: 500x600 пикселей

## Особенности

- Telegram ID начинаются с 100001+ (женщины) и 200001+ (мужчины) для простоты различия
- Все профили активны и видны в ленте
- Все профили заполнены на 100%
- Запуск скрипта автоматически очищает старые данные

## Примечание

Фотографии хранятся как URL в колонке `storage_key` таблицы `profile_photos`. Если ваше приложение использует собственное хранилище (S3, etc), вам может потребоваться адаптировать скрипт для скачивания и загрузки фото.
